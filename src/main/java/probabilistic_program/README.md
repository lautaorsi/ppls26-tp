# Probabilistic Program

This class servers as a user interface so as to make the program easier to use, the currently supported programs are:

- runLWProbabilisticProgram(String program, Random RandomNumberGenerator)

- runSMCProbabilisticProgram(String program, ArrayList < Random > rRandomNumberGenerators, int particle_qtty) 

- runSSMHProbabilisticProgram(String program, Random RandomNumberGenerator, int steps, int warmup_rounds)
