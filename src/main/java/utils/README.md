# Utils

Here are a couple of classes that are needed for things outside of the project's scope


Tuple
---
Simple tuple with arguments first and second

Parser
---
The pogam's parser that transforms a String-type LISP program into a Form-type list

Closure
---
A representation of a Closure, it does not do anything besides storing the data

Primitives
---
A list of primitive operations/keywords

CustomMath
---
Adds the softmax function that pseudonormalizes all values (but by their exponential instead of the actual value)
